მოცემული Android პროექტი წარმოადგენს ვალუტის კონვერტაციის აპლიკაციას, რომელიც შექმნილია MVVM არქიტექტურული პატერნის გამოყენებით. აპლიკაცია უზრუნველყოფს ვალუტების რეალურ დროში კონვერტაციას გარე REST API-ის მეშვეობით და მიღებული შედეგების ლოკალურ ბაზაში შენახვას. პროექტის მიზანია Android-ის ძირითადი არქიტექტურული პრინციპების პრაქტიკული გამოყენება.

აპლიკაციაში მკაცრად არის გამიჯნული UI, ბიზნეს ლოგიკა და მონაცემთა ფენა. ViewModel იყენებს StateFlow / LiveData-ს UI-სთან კომუნიკაციისთვის, ხოლო მონაცემთა წყაროდ გამოყენებულია Repository პატერნი, რომელიც აერთიანებს როგორც Remote (Retrofit), ასევე Local (Room Database) მონაცემებს. ასინქრონული ოპერაციები შესრულებულია Kotlin Coroutines-ის გამოყენებით.

პროექტში განსაკუთრებული ყურადღება დაეთმო კოდის სტრუქტურას, შეცდომების დამუშავებას და მონაცემთა უსაფრთხო მართვას (API key-ის დაცვა BuildConfig-ის საშუალებით). აპლიკაცია სრულად აკმაყოფილებს სასწავლო მოთხოვნებს და დემონსტრირებს Android აპლიკაციის სრულ ციკლს — UI-დან მონაცემთა შენახვამდე.
