­ЪЊў Finance Converter РђЊ Android Project

Finance Converter рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА Android рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЋрЃљрЃџрЃБрЃбрЃўрЃА рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљрЃА, рЃЎрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА.

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ Kotlin-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ MVVM рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃљрЃбрЃћрЃарЃюрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.

­Ъј» рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃБрЃюрЃљрЃарЃћрЃЉрЃўрЃА рЃЊрЃћрЃЏрЃЮрЃюрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃљ:

Android рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃљрЃњрЃћрЃЉрЃљ

MVVM рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
